# Phase 10: Member Event Handlers

## Commit: `feat(events): add member handlers`

## Events
- guildMemberAdd - User joined server
- guildMemberRemove - User left/kicked from server
- guildMemberUpdate - Nickname/roles/timeout changed

## Embed info
- User mention, tag, ID, avatar
- Account creation date
- For join: when they joined
- For leave: how long they were in server
- For update: show what changed (nickname, roles added/removed, timeout)
