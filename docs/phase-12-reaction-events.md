# Phase 12: Reaction Event Handlers

## Commit: `feat(events): add reaction handlers`

## Events
- messageReactionAdd - Reaction added to message
- messageReactionRemove - Reaction removed from message
- messageReactionRemoveAll - All reactions cleared from message
- messageReactionRemoveEmoji - Specific emoji reactions cleared

## Embed info
- User who added/removed reaction
- Message link
- Channel
- Emoji (custom or unicode)
- For removeAll: message info
